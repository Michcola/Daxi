# Daxi - UI 组件

Daxi UI 是一个好用易上手的 UI 框架，提供了一些常用组件，适合 PC 端和移动端使用。

组件：按钮、输入框、网格、布局、Toast、Tabs、Popover、手风琴等

# why Daxi

这是我在学习Vue过程中做的一个UI框架,希望对你有用~
