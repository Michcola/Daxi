# 添加 CSS 样式

在使用本框架前，请在 CSS 中开启 border-box

*，*::before,*::after{ box-sizing: border-box }

IE 8 及以上浏览器支持此样式。

# 引入 daxi

import {Button,ButtonGroup,Icon} from 'daxi'
import 'daxi/dist/index.css';

export default {
  name: 'app',
  components: {
    HelloWorld,
    'g-button':Button
  }
}
